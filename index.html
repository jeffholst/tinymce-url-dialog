TinyMCE URL Dialog

Version: 0.0.1

This demo shows a TinyMCE instance which opens a URL dialog from an external web page. Communication between the TinyMCE and the external web page is handled with the browsers window.postMessage() API which allows sending data across different origins.

To begin click the "Open URL Dialog" button below.
